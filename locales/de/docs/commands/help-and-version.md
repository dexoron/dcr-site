# dcr --help / --version

Top-Level-Flags.

## dcr --help

```bash
dcr --help
```

Zeigt verfügbare Befehle, Flags und Beispiele.

## dcr --version

```bash
dcr --version
```

Gibt Version und Target in folgendem Format aus:

```text
dcr <version> (<target>)
```

## Ausführung ohne Befehl

Wenn `dcr` ohne Argumente ausgeführt wird, zeigt es Hilfe an, entsprechend `dcr --help`.
