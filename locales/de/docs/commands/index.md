# Befehle

Dieser Abschnitt beschreibt alle Befehle, die von der aktuellen DCR-Version unterstützt werden.

## Übersicht

- `dcr new <name>` - neues Projekt in einem Unterverzeichnis erstellen.
- `dcr init` - aktuelles leeres Verzeichnis initialisieren.
- `dcr build [--debug|--release]` - `src/main.c` bauen.
- `dcr run [--debug|--release]` - bauen und `target/<profile>/main` ausführen.
- `dcr clean [--debug|--release]` - `target` vollständig oder profilweise entfernen.
- `dcr --update` - Binärdatei auf das neueste Release aktualisieren.
- `dcr --help` - Hilfe anzeigen.
- `dcr --version` - Version und Target anzeigen.
